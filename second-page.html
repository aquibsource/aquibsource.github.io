BLS Budget Guide

Start with the first Insertion Order

Calculate the IO's per day budget by using this formula: (remaining budget/remaining days) Now, calculate the 10 day budget by doing this: (per day budget)*10

This is the IO's budget for 10 days of flight

Take the first line item

  • If the line item has no budget allocated, no need for this step
  • If the line item has daily budget allocated, then that is the per day budget. So calculate the 10 days budget do this: (per day budget)*10
  • If the line item has some definite budget allocated, substract the already spent budget of this line item and then calculate the remaining budget.
  • Next step is to calculate the per day budget by doing: (remainign budget/remaining days)
  • Take out the 10 day budget by: (per day budget)*10
  • After you've finally got the LI's 10 day budget, note it down.
    Repeat the process for the rest of the Line items in the IO
  • After the budget for all the LIs are found, sum them up
  • Compare this budget with the IO's 10 day budget and take the lower of the budgets from the IO and the total sum of LIs
    • Repeat the same process for all the other Insertion Orders if any!
  • This can get painful. if there are a lot if IOs and LIs. Watch out this space for a tool that would make the process easy.
